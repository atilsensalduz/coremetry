⚠ UPGRADING FROM CHART < 0.1.3?
  The Deployment selector changed (added `app.kubernetes.io/component`)
  to fix a bug where the coremetry Service routed traffic to ClickHouse /
  Redis / OTel Collector pods. Selector is immutable — you must
  uninstall + reinstall instead of `helm upgrade`:
      helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}
      helm install   {{ .Release.Name }} oci://ghcr.io/cilcenk/charts/coremetry --version 0.1.3 -n {{ .Release.Namespace }} ...
  ClickHouse PVC is preserved by default (StatefulSet pattern), so data
  in CH stays. Verify with: kubectl get pvc -n {{ .Release.Namespace }}

1. Wait for the pods to be ready:
     kubectl --namespace {{ .Release.Namespace }} rollout status deploy/{{ include "coremetry.fullname" . }}

2. Access the UI:
{{- if .Values.ingress.enabled }}
{{- range .Values.ingress.hosts }}
     http{{ if $.Values.ingress.tls }}s{{ end }}://{{ .host }}
{{- end }}
{{- else }}
     kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "coremetry.fullname" . }} 8088:{{ .Values.service.httpPort }}
     # then open http://localhost:8088
{{- end }}

3. Sign in with the bootstrap admin (rotate the password immediately):
     email:    {{ .Values.config.auth.initialAdmin }}
     password: (the value of secrets.initialAdminPassword — default "admin")

4. Point your apps at the OTel Collector:
{{- if .Values.otelCollector.enabled }}
     OTEL_EXPORTER_OTLP_ENDPOINT=http://{{ include "coremetry.fullname" . }}-otelcol:{{ .Values.otelCollector.service.grpcPort }}
{{- else }}
     # Collector disabled — point apps directly at coremetry's gRPC service:
     OTEL_EXPORTER_OTLP_ENDPOINT=http://{{ include "coremetry.fullname" . }}:{{ .Values.service.grpcPort }}
{{- end }}

{{- if $redisURL }}
   Redis is wired ✓ — every background worker uses a per-tick Redis
   lock so scaling out is safe. To run N replicas:
       helm upgrade {{ .Release.Name }} --set replicaCount=N ...
   View the live roster at /admin/cluster (admin role required) —
   each pod heartbeats every 10s, stale-out after 30s.
